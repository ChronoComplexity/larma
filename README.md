# larma

Pet cozy game with a gamified alarm mechanic. Rewards for waking up and not snoozing your alarm, verified via a phone call with a 'pet manager' style character (with the option of saying I want to go back to sleep call me back later), no rewards for sleeping in. Made with a Typescript backend and a Typescript/React frontend.
